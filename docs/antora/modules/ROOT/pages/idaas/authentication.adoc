= 身份认证

本文讨论身份认证的实现思路。

身份认证有很多种实现方式，下面逐一讨论。

== 实现原理

事先收集用户信息（RegisteredUser），提供认证接口，登录时对比输入的用户信息（LoginedUser）和 RegisteredUser 是否匹配。

== 实现方式

=== 用户名 + 密码

用户通过输入 *用户名* 和 *密码* 登录系统，*用户名* 也可以是手机号、邮箱或其他可以唯一标识用户的字段。
